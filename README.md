# Credit Risk Analysis 

An end-to-end data analytics project focused on identifying factors influencing loan defaults and borrower risk. The project combines Python-based exploratory data analysis with an interactive Power BI dashboard to uncover patterns in loan grades, income levels, interest rates, home ownership, and previous default history.

# Key Insights:
• Overall Default Rate: 21.82%
• Grade G loans recorded the highest default rate (98.44%)
• Debt Consolidation loans showed the highest risk (28.59%)
• Previous defaulters were more than twice as likely to default again
• Lower-income borrowers exhibited significantly higher default rates

# Tools Used:
Python, Pandas, NumPy, Matplotlib, Seaborn, Power BI, DAX

Data Cleaning • EDA • Data Visualization • DAX • Business Intelligence • Risk Analytics
